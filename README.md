# Currency Converter
**Автор:** Акиншина Анна

## Описание
Приложение для конвертации валют в реальном времени.

## Как получить API-ключ
1. Зарегистрируйтесь на [ExchangeRate-API](https://exchangerate-api.com).
2. Скопируйте ваш персональный API Key.
3. Вставьте его в переменную `API_KEY` в коде.

## Использование
1. Запустите скрипт: `python main.py`.
2. Введите сумму, выберите валюты и нажмите кнопку.
3. История сохранится автоматически.
